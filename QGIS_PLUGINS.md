# Recommended QGIS Plugins for Kensington Project
**Target Workflow:** Database -> Analysis -> Professional Output

## 1. QuickMapServices
- **Use:** Provides high-quality base maps (OpenStreetMap, Google, Esri) for context.
- **Why:** Essential for orienting your MTM10 data.

## 2. QGIS2Web
- **Use:** Automatically exports your QGIS map to an interactive web-ready OpenLayers or Leaflet map.
- **Why:** The easiest way to share interactive results without writing code.

## 3. Data Plotly
- **Use:** Integrates D3.js charts directly into QGIS.
- **Why:** Create demographic and infrastructure charts from your SQL attribute tables.

## 4. Layout Manager / Atlas
- **Use:** Built-in tool for creating print-ready maps (PDF/PNG).
- **Why:** Automation! Create one layout and generate 100 maps using the 'Atlas' feature.

## 5. DB Manager (Core Plugin)
- **Use:** Directly query your SpatiaLite database.
- **Why:** Perform SQL queries from within the QGIS interface.
